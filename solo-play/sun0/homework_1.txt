
1. 웹 브라우저의 종류

<마이크로소프트 엣지 브라우저>

: window 10 에 탑재됨. 

: 2010년대 이후로 인터넷 익스플로러의 시장 점유율이 하락하면서, 마이크로소프트에서 새로이 출시한 브라우저.

: 인터넷 익스플로러의 문제점으로 지적되던 "Active X" 를 덜어내고 좀 더 가볍게 코드 설계를 바꾸었다고 함.

: 다른 브라우저들의 점유율이 이미 높아져있어서 그런지 시장에서 많이 사용하지 않음..


<마이크로소프트 인터넷 익스플로러>

: 마이크로소프트의 브라우저로 window 에 기본 탑재되어 있어 초기 시장 점유율이 높았음.

: 최근에는 다른 웹 브라우저들의 출시와 보안적 취약함으로 인해 점유율이 낮아진 상태.

: 현재 권장하지 않는 브라우저.


<구글 크롬>

: 구글에서 출시한 웹 브라우저로 자사 검색엔진에 최적화되어 있음.

: Windows 뿐만 아니라 MAC OS, Linux, 안드로이드, iOS 등지에도 지원하는 범용적인 브라우저.

: 인터넷 익스프로러의 점유율 하락되며 동시에 크롬으로 점유율이 대거 넘어감.

: 현재 가장 인기있는 웹 브라우저 중 1. 대중적으로 사용됨.

: Active X 호환성도 많이 개선되어, 은행 및 결제 사이트 접속도 무리없이 가능함,

:  생각보다 보안에 취약하고 시스템 자원을 많이 소모함. (탭 기능이 각 탭을 열게되면 탭이 일종의 프로세스로 활성화 되면서 시스템 메모리를 많이 소모함)


<모질라 파이어폭스>
 
: 저사양의 PC 에서도 빠르다는 특징. 출시된 웹 브라우저중 빠르다는 평가를 받음.

: 넷스케이프의 소스를 모질라 재단에서 인수했으며, 이를 리비전 하면서 고유 엔진인 "GECKO" 엔진 개발함.

: 보편적인 OS 거의 다 지원함. (Windows, Linux, MAC OS, 안드로잊드, iOS)

: 크롬, 사파리 뒤를 잇는 점유율을 가짐.


<애플 사파리>

: 애플 사에서 개발한 웹 브라우저로 범용적인 브라우저라기 보다는 MAC OS, iOS 전용에 가까움.

: 전 세계적으로 MAC 과 아이폰의 점유율이 상당하기에 전체 시장 점유율이 높음.

: 오픈 소스 엔진인 "Webkit" 엔진 기반으로 개발됨.

: MAC OS 내에서도 메모리를 많이 점유하나, MAC OS 내의 다른 웹 브라우저에 비해서는 메모리 누수가 적고 안정적으로 동작하기에 애플 제품에서는 
사파리를 쓰는게 괜찮음.

: 타 OS 에 출시가 안되는 폐쇄성은 아쉬운 부분.


<삼성 인터넷 브라우저>

: 삼성전자의 스마트폰에 기본적으로 탑재되어 있는 모바일 용 웹 브라우저

: 크롬 기반의 블링크 엔진 탑재했으며, 웹 로딩이 빠르고 가볍고 안정적인 모습을 보여줌.

: 삼성전자의 스마트폰이 워낙 잘 팔리기 때문에 점유율이 높음.


2. 어떤 웹 브라우저를 사용하는게 좋을까

사용성에 따라 다른듯..

일반 사용자(웹 서핑, 쇼핑, 뱅킹) 라면? 구글 크롬 추천

-> 쇼핑몰의 결제나 뱅킹 대부분 가능. PC에서 사용하기 무난함.

저사양 PC 의 소유자라면? 파이어폭스 혹은 오페라 추천

디자이너 등의 그래픽 관련 업무를 한다면? 사파리 추천
-> 디자인 관련 프로그램이나 툴 사용 시 애플 Mac 을 주로 사용하는 경향.. 이때에는 사파리 사용이 최적.

웹 개발 및 프로그래머라면? 구글 크롬 추천
-> 개발자 도구와 개발을 위한 확장 프로그램이 즐비함. 개발을 편리하게 할 수 있음.
-> 문서를 찾거나 여러 탭을 써야하는경우 파이어폭스도 좋으 선택. 문서를 로드하기에 최적화된 브라우저환경 제공함.
