<정적 웹페이지>
- 웹 서버에 저장된 html 문서를 그대로 클라이언트에게 전달하는 웹 페이지
- 서버에 저장된 데이터가 변경되지 않는 한 고정된 웹 페이지를 보게됨
- 모든 사용자는 같은 결과의 웹 페이지를 서버에 요청하고 응답 받음

예시 : 회사 소개 페이지 (http://btgin.co.kr/#sec-9)

<동적 웹페이지>
- 웹 서버에서 요청 정보를 추가적으로 처리한 후에 페이지를 클라이언트에게 전달
- 웹 페이지의 특정 부분이 동적으로 변할 수 있음. (상품 검색, 뉴스 댓글 등..)
- 사용자는 시간, 상황, 요청에 따라 다른 웹 페이지를 보게됨
- 사용자마다 다른 결과의 웹 페이지를 서버에 요청하고 응답 받을 수 있음
- 우리가 보는 대부분의 웹 페이지

예시 : 포털 메인 (https://www.naver.com/)
